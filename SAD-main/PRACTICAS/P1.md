
# ESTUDIO DE LAS PRINCIPALES EXTENSIONES EN VSCODE

### PHP, HTML, CSS y JS
* HTML Snipetts
  - Extensión: HTML Snippets
HTML Snippets es una extensión popular para editores de código, como Visual Studio Code, que proporciona fragmentos de código predefinidos (snippets) de HTML para facilitar y acelerar el proceso de desarrollo web. Esta extensión ayuda a los desarrolladores a insertar rápidamente estructuras HTML comunes mediante atajos de teclado o comandos, lo que optimiza el flujo de trabajo.
La extensión te permite escribir menos código al proporcionar plantillas de código HTML. En lugar de escribir manualmente etiquetas y atributos repetitivos, puedes escribir abreviaturas que la extensión expandirá automáticamente a fragmentos HTML completos.

* JShint
  - Errores de sintaxis:
Falta de comas, puntos y comas, llaves o paréntesis mal balanceados, entre otros errores sintácticos.
  - Prácticas de codificación inseguras:
Código que podría causar comportamientos no esperados o ser inseguro, como el uso indebido de variables globales o construcciones que pueden generar bugs.
  - Violaciones de estilo de codificación:
Puede verificar si el código sigue ciertos estilos de escritura o convenciones, como el uso de comillas simples o dobles, la indentación adecuada, y más.
  - Advertencias sobre compatibilidad:
JSHint puede avisar sobre características del lenguaje que podrían no ser compatibles con todos los navegadores o entornos de ejecución.

* PHP Debug
  + PHP debug es una de las herramientas más poderosas para depuración interactiva en PHP. Ofrece características avanzadas como:
  * Detección de errores: PHP debug mejora los mensajes de error y muestra un stack trace más detallado.
  * Puntos de interrupción (breakpoints): Permite pausar la ejecución del código en un punto específico y examinar variables y estados en tiempo real.
  * Perfilado: Mide el rendimiento de la aplicación, ayudándote a identificar cuellos de botella.
  * Depuración remota: Con Xdebug puedes depurar código en un entorno de servidor remoto desde tu máquina local, usando un editor como Visual Studio Code, PHPStorm, entre otros.

  * CSS Beautify
  + CSS Beautify toma código CSS que puede estar desordenado, minificado, o difícil de leer y lo reformatea de manera consistente. Esto incluye:
  - Indentación correcta:
Alinea el código CSS con una estructura jerárquica clara, usando indentaciones uniformes (espacios o tabulaciones) para que sea más fácil identificar las reglas y propiedades dentro de los selectores.
  - Organización de propiedades:
Reorganiza las propiedades de cada regla CSS en líneas separadas, a menudo en un orden alfabético o según convenciones establecidas.
  - Espaciado adecuado:
Asegura que haya un espaciado uniforme entre los selectores, las llaves, y las propiedades para mejorar la claridad del código.
Limpieza de comentarios y código redundante:

- En algunos casos, puede eliminar o resaltar comentarios innecesarios y eliminar código CSS repetitivo o sin uso.
