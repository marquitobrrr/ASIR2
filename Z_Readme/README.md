## 🌟 Hitos de Administración y Sistemas (ASIR)

> [!IMPORTANT]
> **Enfoque del Repositorio:** Gestión integral de servicios de red, securización de entornos críticos y despliegue de infraestructuras escalables en la nube.

* **🛡️ Hardening & Pentesting:** Implementación de auditorías de seguridad usando entornos controlados como **DVWA** y análisis de malware con **Cuckoo Sandbox**.
* **☁️ Cloud & Web Deployment:** Configuración de arquitecturas de alta disponibilidad en **AWS** y despliegue optimizado de aplicaciones web (LAMP/WAMP).
* **📊 Big Data & DB Admin:** Administración avanzada de bases de datos relacionales y NoSQL, garantizando la integridad, replicación y alta disponibilidad de los datos.
* **🔒 Network Security:** Configuración avanzada de firewalls con **pfSense**, gestión de diccionarios con **Pydictor** y ataques de fuerza bruta controlados con **Hydra**.
